# Rachel's Resume

## Description:
This is a one page webpage that displays Rachel's Portfolio. This project included one html file index.html and one css file styles.css.

## Usage:
The buttons handshake, LinkedIn, Github, and Goodreads do transport the user to profiles on those websites. The resume button leads
to a download pdf version of the resume and the email button will lead you to send an email. There are skip links at the top to easily jump around.
